# IR-Receiver
Dynamic Infrared Receiver, saving sendt signals and assigning them to actions.

Supporting NEC protocol

| Signal        | Bytes | Use  |
| ------------- |:-----:| -----:|
| Init          | 8     | - |
| Address       | 8     | - |
| Inverted Addr | 8     | - |
| Command       | 8     | - |
| Inverted Comm | 8     | - |
